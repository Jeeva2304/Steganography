# Steganography
Image and Audio Steganography in Python

GUI used Tkinter. It is the simple GUI library in Python.
Steganography.py is the main code which imports and call all other python files.
This python program hides a text message into an image or into an audio files based on the user's choice. It also extract the information from the stego-file and displayed the user. It uses Least Significant Bit (LSB) Algorithm from embedding ansd extraction processes.
